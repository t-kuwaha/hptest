アプリケーション

Code for Kitakyushuの取組みによって作成したアプリケーションを紹介しています。

飲食店向けスマホサイトテンプレート

無料で利用可能な飲食店向けスマホサイトのテンプレート（ひな形）を提供しています。

北九州テイクアウトマップ

北九州市内（近隣含む）でテイクアウト可能な飲食店を掲載しています。

北九州市新型コロナウイルス感染症対策サイト

北九州市内の最新感染動向について、北九州市が発表したオープンデータを見やすく表示しています。
サイト制作は株式会社ラック様で、Code for Kitakyushuは運営に協力しています。
